GetTriEdgeFaces

Short Description   

Get all the surfaces having three edges and put them in separated groups.

Additional Information   

-

Definition   

def GetTriEdgeFaces( shape = None , tol = 1e-7 , add = True ):

Argument(s)   

Name Description Type GUI selection [?] Selection by name [?] Recursive [?] Default value
shape The shape in which to look for tri-edge faces. Any geometrical object yes yes yes None
tol See here. Float - - - 1e-7
add See here. Boolean - - - True

Returned Value(s)   

dim Value [?] single Value [?] Type Number Name
- - Group of Faces n "TriEdgeFace"

Example   

from cfdmsh import * # To adapt to the cfdmsh installation method

box = geompy.MakeBoxDXDYDZ(100, 100, 100)

vertex1 = geompy.MakeVertex(0, 100, 100)
vertex2 = geompy.MakeVertex(100, 0, 100)
vertex3 = geompy.MakeVertex(100, 0, 0)

plane = geompy.MakePlaneThreePnt(vertex1, vertex2, vertex3, 300)

partition = geompy.MakePartition([box], [plane])
AddToStudy(partition, "partition")

tri_edge_faces = GetTriEdgeFaces(partition)
