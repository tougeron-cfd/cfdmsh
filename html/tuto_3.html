Tutorial 3

The aim of this tutorial is to show some techniques of face rebuilding using cfdmsh.

Face rebuilding is particularly useful when imported geometries contained small faces, sharp angles or undesired vertexes which can dramatically impact the surface mesh quality.

The following cfdmsh functions will be used:

  • FuseShellFaces
  • FuseSplines
  • SetRandomColors
  • RebuildFace
  • FuseSplineSets
  • MakeFillingFromUnsortedEdges
  • UnrefineSplineSet
  • RebuildSpline
  • SwitchSplineSet

NB: This tutorial was tested on Salome 7.7.1 on Linux with the following settings:

File > Preferences... > Geometry > Default display mode > Wireframe.

File > Preferences... > Geometry > Hide input objects from viewer > OFF.

Content of this tutorial:

  • Rebuild Nose Faces
  • Rebuild Coinciding Faces
  • Rebuild Non-Coinciding Faces
