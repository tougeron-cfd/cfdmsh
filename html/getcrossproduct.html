GetCrossProduct

Short Description   

Computes the cross product between two 3D vectors.

Additional Information   

-

Definition   

def GetCrossProduct(vector_1, vector_2, tol = 1e-7, add = True):

Argument(s)   

Name Description Type GUI selection [?] Selection by name [?] Recursive [?] Default value
vector_1 The first vector. Vector - yes - None
vector_2 The second vector. Vector - yes - None
tol See here. Float - - - 1e-7
add See here. Boolean - - - True

Returned Value(s)   

dim Value [?] single Value [?] Type Number Name
- - Vector 1 "CrossProduct"

Example   

from cfdmsh import * # To adapt to the cfdmsh installation method

v1 = geompy.MakeVertex(0, 0, 0)
v2 = geompy.MakeVertex(0.2, 0.3, 0.1)
v3 = geompy.MakeVertex(0, 0.3, -0.25)

vector_1 = geompy.MakeVector(v1, v2)
vector_2 = geompy.MakeVector(v1, v3)
AddToStudy([vector_1, vector_2], "vector")

cross_product = GetCrossProduct(vector_1, vector_2)
