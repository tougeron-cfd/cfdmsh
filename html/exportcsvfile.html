ExportCSVFile

Short Description   

Exports a 3D vertex compound into a CSV file.

Additional Information   

-

Definition   

def ExportCSVFile( compound = None, file = None, head = True ):

Argument(s)   

Name Description Type GUI selection [?] Selection by name [?] Recursive [?] Default value
compound The vertex compound to export. Compound of Vertexes yes yes - None
file The name of the file to write. String - - - None
head Defines if the function has to write a header to the file. Boolean - - - True

Returned Value(s)   

dim Value [?] single Value [?] Type Number Name
- - - - -

Example   

from cfdmsh import * # To adapt to the cfdmsh installation method

box = geompy.MakeBoxDXDYDZ(100, 100, 100)
box_faces = GetSubShapes(box)[2]
vertex_compounds = RebuildFace(5, box_faces, dim = 0, add = False)
compound = geompy.MakePartition(vertex_compounds)
AddToStudy(compound, "compound")

ExportCSVFile(compound, "csv_file_example")
ImportCSVFile("csv_file_example")
ImportCSVFile("csv_file_example", single = False)
