PutAllSubShapesInAGroup

Short Description   

Create a geometrical group containing all sub-shapes of a given dimension.

Additional Information   

-

Definition   

def PutAllSubShapesInAGroup( dim, shape = None, add = True, infa = True ):

Argument(s)   

Name Description Type GUI selection [?] Selection by name [?] Recursive [?] Default value
dim See here. Integer - - - -
shape The source shape. Any geometrical object yes yes yes None
add See here. Boolean - - - True
infa See here. Boolean - - - True

Returned Value(s)   

dim Value [?] single Value [?] Type Number Name
0 - Group of Vertexes 1 "AllSubShapes (Vertexes)"
1 - Group of Edges 1 "AllSubShapes (Edges)"
2 - Group of Faces 1 "AllSubShapes (Faces)"
3 - Group of Solids 1 "AllSubShapes (Solids)"

Example   

from cfdmsh import * # To adapt to the cfdmsh installation method

box = geompy.MakeBoxDXDYDZ(100, 100, 100)
AddToStudy(box, "box")

group_1 = PutAllSubShapesInAGroup(0, box)
group_2 = PutAllSubShapesInAGroup(1, box)
group_2 = PutAllSubShapesInAGroup(2, box)
