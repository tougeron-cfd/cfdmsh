GetSubShapes

Short Description   

Gets all sub-vertexes, sub-edges, sub-faces and sub-solids of a geometrical shape.

Additional Information   

The function returns a list of lists, each one containing all sub-shapes of a given dimension.

Variable Description
returned_list[0] List of 0D sub-shapes (vertexes)
returned_list[1] List of 1D sub-shapes (edges)
returned_list[2] List of 2D sub-shapes (faces)
returned_list[3] List of 3D sub-shapes (volumes)
returned_list[-1] The input shape itself

Definition   

def GetSubShapes(shape):

Argument(s)   

Name Description Type GUI selection [?] Selection by name [?] Recursive [?] Default value
shape The shape from which to get sub shapes. Any geometrical object - - - -

Returned Value(s)   

dim Value [?] single Value [?] Type Number Name
- - Geometrical object or List of Geometrical objects (see the "Additional Information") 5 -

Example   

from cfdmsh import * # To adapt to the cfdmsh installation method

box = geompy.MakeBoxDXDYDZ(10, 10, 10)

box = GetSubShapes(box)

AddToStudy(box[-1], "This is the input shape")
AddToStudy(box[0], "Sub-Vertex")
AddToStudy(box[1], "Sub-Edge")
AddToStudy(box[2], "Sub-Face")
