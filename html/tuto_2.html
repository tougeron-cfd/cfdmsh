Tutorial 2

The aim of this tutorial is to give the basis of viscous layer blocking generation and meshing using cfdmsh.

The following cfdmsh functions will be used:

  • MakeFoilFromUnsortedVertexes
  • MakePlanarWireOffset
  • ExtendViscousLayer
  • MakeLinkingSolids
  • MakeTipViscousLayer
  • ExtendTipViscousLayer
  • SetRandomColors
  • ProjectEdgeSubmesh
  • ViscousLayerScaleFactor
  • PutAllSubShapesInAGroup
  • GetTriEdgeFaces
  • MakeTriEdgeFaceSubmeshes

NB: This tutorial was tested on Salome 7.7.1 on Linux with the following settings:

File > Preferences... > Geometry > Default display mode > Wireframe.

File > Preferences... > Geometry > Hide input objects from viewer > OFF.

Content of this tutorial:

  • Create the Wing Blocking
  • Create the Tip Blocking
  • Generate the 1D Mesh
  • Generate the 3D Mesh
