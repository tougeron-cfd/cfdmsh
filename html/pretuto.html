Preliminary Tutorial

The purpose of this tutorial is to give basic knowledge about the manipulatio of the Salome GUI as well as to give example of fast interaction with its Python console thanks to cfdmsh.

The following cfdmsh functions will be used:

  • PrintDefinedFunctions
  • GetGUISelection
  • GetSubShapes
  • AddToStudy

NB: This tutorial was tested on Salome 7.7.1 on Linux with the following settings:

File > Preferences... > Geometry > Default display mode > Wireframe.

File > Preferences... > Geometry > Hide input objects from viewer > OFF.

Content of this mini-tutorial:

  • Open a New Study
  • Manipulate the 3D Window
  • Load Cfdmsh
  • Interact With the Python Console

Open a New Study   

  1. Open Salome.
  2. Open a new study.

    3. File > New

  3. Load the Geometry module.

    4. Click on in the toolbar

  4. Expand the Geometry node in the study tree.

Manipulate the 3D Window   

  1. Create a cube.

    2. New Entity > Primitives > Box → Apply and Close

  2. Get comfortable with the 3D window:
    3. Action Keys
    Zoom [Ctrl] + Left button
    Rotate [Ctrl] + Right button
    Rotate perpendicular to the screen [Ctrl] + Right button far from the center of the 3D window
    Pan [Ctrl] + Middle button
  3. Explode this cube into sub-faces.

    4. Select "Box_1" → New Entity > Explode →

    Set "Sub-shapes Type" to "Face" → Apply and Close

  4. Show only the exploded faces.

    5. Select "Box_1" → Right-click > Show Only Children

  5. Get comfortable with the 3D window selection:
    6. Action Keys
    Select one entity Left button
    Select several entities [Shift] + Left button * n
    or
    Left button + drag
    or
    Right button [+ drag + Left button] * n

Load Cfdmsh   

  1. Load the cfdmsh library.

    2. [Ctrl] + [T] → Select the "cfdmsh.py" file on your computer

  2. Display all implemented cfdmsh functions.

    3. In the Python console, type:

    PrintDefinedFunctions( )

    or:

    pdf( )

  3. You can also display them in condensed mode:

    4. In the Python console, type:

    PrintDefinedFunctions( 1 )

    or:

    pdf( 1 )

  4. Display the help documentation for the function RebuildFace.

    5. In the Python console, type:

    help( RebuildFace )

    or:

    help( rf )

Interact With the Python Console   

  1. Put the cube in a python variable called "box".

    2. Select "Box_1" → In the Python console, type:

    box = GetGUISelection( )

    or:

    box = ggs( )

  2. Display all methods associated to this shape.

    3. In the Python console, type this without pressing [Return]:

    box.Get

    → press [Tab] → Scroll down your Python console using your mouse wheel

  3. Get all sub-solids, sub-faces and sub-vertexes of the box in one command.

    4. In the Python console, type [Esc], then:

    sub_shapes = GetSubShapes( box )

    or:

    sub_shapes = gss( box )

    The object returned by the GetSubShapes function is a list organized as following:

    Variable Description
    sub_shapes[0] List of 0D sub-shapes (vertexes)
    sub_shapes[1] List of 1D sub-shapes (edges)
    sub_shapes[2] List of 2D sub-shapes (faces)
    sub_shapes[3] List of 3D sub-shapes (volumes)
    sub_shapes[-1] The main shape itself (here, the box)
  4. Publish all sub-vertexes in the Salome study.

    5. In the Python console, type:

    AddToStudy( sub_shapes[0] )

    or:

    ats( sub_shapes[0] )

  5. Publish all sub-edges in the Salome study, calling them "Edge".

    6. In the Python console, type:

    AddToStudy( sub_shapes[1], "Edge" )

    or:

    ats( sub_shapes[1], "Edge" )

    You can close this study.
