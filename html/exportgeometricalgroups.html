ExportGeometricalGroups

Short Description   

Exports into a file the geometrical groups of a geometrical object in the form of sets of subshape IDs.

Additional Information   

This function writes in a file the name, type and subshape IDs of each group of a geometrical object. The file can then be imported with the ImportGeometricalGroups function.

The file appears by default in the home folder.

Definition   

def ExportGeometricalGroups( shape = None, file = "cfdmsh_grps", only = [None], ignore = [None], type = None ):

Argument(s)   

Name Description Type GUI selection [?] Selection by name [?] Recursive [?] Default value
shape The source geometrical object. Any geometrical object yes yes - None
file The name of the file to write. String - - - "cfdmsh_grps"
only The list of names of groups to export, excluding the others. List of Strings - - - [None]
ignore The list of names of groups to ignore. List of Strings - - - [None]
type Type of groups to export. Can equal "vertex", "edge", "face" or "solid". String - - - None

Returned Value(s)   

dim Value [?] single Value [?] Type Number Name
- - - - -

Example   

from cfdmsh import * # To adapt to the cfdmsh installation method

box1 = geompy.MakeBoxDXDYDZ(100, 100, 100)
box2 = geompy.MakeBoxDXDYDZ(100, 100, 100)
AddToStudy([box1, box2], "Box_1")

group1 = geompy.CreateGroup(box1, geompy.ShapeType["VERTEX"])
geompy.AddObject(group1, 21)
geompy.AddObject(group1, 11)

group2 = geompy.CreateGroup(box1, geompy.ShapeType["EDGE"])
geompy.AddObject(group2, 5)
geompy.AddObject(group2, 18)

group3 = geompy.CreateGroup(box1, geompy.ShapeType["FACE"])
geompy.AddObject(group3, 3)
geompy.AddObject(group3, 33)

AddToStudy([group1, group2, group3], "group", father = box1)

ExportGeometricalGroups(box1, "group_file_example")
ImportGeometricalGroups(box2, "group_file_example")
