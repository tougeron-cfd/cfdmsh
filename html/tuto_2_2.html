Tutorial 2

Create the Tip Blocking   

Content of this section:

  • Create the 3D Wing Tip Blocking
  • Extend the 3D Wing Tip Blocking
  • Create the Final 3D Blocking Object

Create the 3D Wing Tip Blocking   

  1. Explode from the linking solids sub-edges close to the wing tip.

    2. Select "LinkingSolids_1" → New Entity > Explode →

    Set "Sub-shapes Type" to "Edge" → Activate the "Select Sub-shapes" option →

    Select the edges like in the following picture → Apply and Close

  2. Show only the exploded edges.

    3. Select "LinkingSolids_1" → Right-click > Show Only Children

  3. Get the name on the upper offset edge (eg. "Edge_20").

  4. Make a 3D viscous layer block having a thickness of 3cm on the wing tip upper edge.

    5. Select the wing tip upper edge like in the following picture →

    Type in the Python console (replacing "Edge_20" with the suitable name):

    MakeTipViscousLayer( 0.03, "Edge_20" )

    or:

    mtvl( 0.03, "Edge_20" )

  5. Repeat this process to create a 3D viscous layer block on the wing tip lower edge.
  6. Show once again only the previously exploded edges.

    7. Select "LinkingSolids_1" → Right-click > Show Only Children

  7. Create a face from the wing tip.

    8. Select the following three edges →

    New Entity > Build > Face → Apply and Close

  8. Extrude the wing tip face 3cm following Z.

    9. Select the wing tip face ("Face_1") →

    New Entity > Generation > Extrusion → Select the icon →

    Set "Dz" to 0.03" → Apply and Close

  9. Put all tip viscous layer volumes into a compound.

    10. Select "TipViscousLayer_1" and "TipViscousLayer_2" + "Extrusion_1" →

    New Entity > Build > Compound → Apply and Close

  10. Show only this compound.

    11. Select "Compound_2" → Right-click > Show Only

Extend the 3D Wing Tip Blocking   

  1. Create a face group on the compound containing trailing its faces.

    2. Select "Compound_2" → New Entity > Group > Create Group →

    Select the icon → Select the following faces → Click on "Add" →

    Apply and Close

  2. Show again only the linking solid sub-edges.

    3. Select "LinkingSolids_1" → Right-click > Show Only Children

  3. Create a compound from edges being behind the wing tip trailing edge.

    4. Select the edges like in the following picture →

    New Entity > Build > Compound → Apply and Close

  4. Extend the 3D tip viscous layer.

    5. Select the face group "Group_3" and the edge compound "Compound_3" →

    Type in the Python console:

    ExtendTipViscousLayer( )

    or:

    etvl( )

Create the Final 3D Blocking Object   

  1. Put all 3D blocks in a single compound.

    2. Select the "LinkingSolids_1", "Compound_2" and "TipViscousLayerExtension_1" →

    New Entity > Build > Compound → Apply and Close

  2. Progressively "glue" faces inside this compound.

    3. Select the "Compound_4" → Repair > Glue Faces → Apply →

    Select the "Glue_1" → Set "Tolerance" to 1e-6 → Apply →

    Select the "Glue_2" → Set "Tolerance" to 1e-5 → Apply →

    Select the "Glue_3" → Set "Tolerance" to 1e-4 → Apply →

    Select the icon → Set "Tolerance" to 1e-3 →

    Click on "Detect [Glue faces]" → Check there is no face to glue → Close

  3. Show only the final blocking and display it in shading mode.

    4. Select "Glue_4" → Right-click > Show Only

    Select "Glue_4" → Right-click in the 3D window > Display Mode > Shading With Edges
