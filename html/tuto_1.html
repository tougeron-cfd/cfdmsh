Tutorial 1

The aim of this tutorial is to show how to quickly generate a high quality 2D mesh around a simple naca foil, including viscous layers.

In this tutorial, the user takes advantage of the automatic viscous layer generation tool provided by Salome, which can be smartly combined with cfdmsh functions (as we will see).

The following cfdmsh functions will be used:

  • ImportCSVFile
  • MakeFoilFromUnsortedVertexes
  • MakeFoilTrailingFillets
  • MakeVirtualOffsetEdgeSubmeshes

NB: This tutorial was tested on Salome 7.7.1 on Linux with the following settings:

File > Preferences... > Geometry > Default display mode > Wireframe.

File > Preferences... > Geometry > Hide input objects from viewer > OFF.

Content of this tutorial:

  • Prepare the Geometry
  • Generate the Mesh
