ViscousLayerScaleFactor

Short Description   

Calculates the parameters to use in a "Nb. Segment" hypothesis from common viscous layers parameters (total thickness, wall thickness and a ratio).

Additional Information   

The function prints and returns suitable values to put in the "Number of Segments" and "Scale Factor" fields appearing in "Nb. Segment" hypotheses when using a "Scale distribution".

Definition   

def ViscousLayerScaleFactor( total_thick, wall_thick, ratio = 1.2 ):

Argument(s)   

Name Description Type GUI selection [?] Selection by name [?] Recursive [?] Default value
total_thick The viscous layer total thickness. Float - - - -
wall_thick The desired thickness of the layer touching the wall. Float - - - -
ratio The desired ratio between each layer n and n+1. Float - - - 1.2

Returned Value(s)   

dim Value [?] single Value [?] Type Number Name
- - Float 2 -

Example   

from cfdmsh import * # To adapt to the cfdmsh installation method

[nb_segments, scale_factor] = ViscousLayerScaleFactor(0.01, 1e-6, 1.2)
