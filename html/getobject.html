GetObject

Short Description   

Gets a published object, according to its name in the study tree and its component.

Additional Information   

The function first checks if the input object is a string or not. If not, the object is returned as it is. If so, the function looks for the given object in the study tree. If the object is not found, an error message is displayed and the function returns "error".

Definition   

def GetObject(object = None, comp = "GEOM", silent = False):

Argument(s)   

Name Description Type GUI selection [?] Selection by name [?] Recursive [?] Default value
object The name of the object to get. String - - - -
comp The name of the component : "GEOM" or "SMESH". String - - - "GEOM"
silent Determines if the error message, in case no object was found, has to be hidden or not. Boolean - - - False

Returned Value(s)   

dim Value [?] single Value [?] Type Number Name
- - Any object 1 -

Example   

from cfdmsh import * # To adapt to the cfdmsh installation method

geompy.MakeVertex(0, 0, 0, theName = "Vertex_1")

Vertex_1 = GetObject("Vertex_1")
print Vertex_1
