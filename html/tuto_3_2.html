Tutorial 3

Rebuild Coinciding Faces   

Content of this section:

Prepare the Geometry File   

  1. Go to the download page of this documentation and save the files a380-stabilizer.brep in your home folder.

Open a New Study   

  1. Open Salome.
  2. Open a new study.

    3. File > New

  3. Load the Geometry module.

    4. Click on in the toolbar

  4. Expand the Geometry node in the study tree.

  5. Load the cfdmsh library.

    6. [Ctrl] + [T] → Select the "cfdmsh.py" file on your computer

Import the Geometry   

  1. Import the a380-stabilizer.brep file which is in your home folder.

    2. File > Import > BREP

  2. Zoom on the imported geometry.

    3. Click on in the 3D window toolbar

  3. Hide the trihedron.

    4. Click on in the 3D window toolbar

Explode Sub-Faces   

  1. Get all sub-faces from the imported geometry.

    2. Select "a380-stabilizer.brep_1" → New Entity > Explode →

    Set "Sub-shapes Type" to "Face" → Apply and Close

  2. Show only the exploded faces.

    3. Select "a380-stabilizer.brep_1" → Right-click > Show Only Children

Create Spline Sets   

  1. Show only the faces at the tip of the geometry, like indicated in the following picture.

    2. Select the faces indicated in the following picture → Right-click > Show Only

  2. Hide the face closing the geometry tip.

    3. Select the face indicated in the following picture → Right-click > Hide

  3. Create spline sets on each visible face.

    4. Select all visible faces → Type in the Python console:

    RebuildFace( dim = 1 )

    or:

    rf( dim = 1 )

  4. Show only the spline sets, using different colors.

    5. Select all "RebuildFace (Edges)_*" shapes → Right-click > Show Only

    Select all "RebuildFace (Edges)_*" shapes → Type in the Python console:

    SetRandomColors( )

    or:

    src( )

Fuse Spline Sets   

  1. Fuse the indicated spline sets.

    2. Select the spline sets indicated in the following picture →

    Type in the Python console:

    FuseSplineSets( )

    or:

    fss( )

  2. Hide fused spline sets.

    3. Select the spline sets which were fused (eg. like in the previous picture) → Right-click > Hide

  3. Fuse the last resulting shape ("FusedSplineSets_1") with the indicated spline set.

    4. Select the last resulting shape (eg. "FusedSplineSets_1") as well as the spline set indicated in yellow in the following picture →

    Type in the Python console:

    FuseSplineSets( )

    or:

    fss( )

  4. Repeat steps 6 and 7 to fuse the last resulting shape ("FusedSplineSets_2") with the spline set indicated in green in the previous picture, using this time 200 points to fuse splines and a tolerance of 1e-4:

    5. FuseSplineSets( np = 200, tol = 1e-4 )

    or:

    fss( np = 200, tol = 1e-4 )

  5. Show only the last resulting shape.

    6. Select "FusedSplineSets_3" → Right-click > Show Only

Build a New Face   

  1. Create a vertex at a corner of this spline set.

    2. New Entity > Basic > Point → Select the vertex visible in the following picture →

    Apply and Close

  2. Create a new face from this spline set, starting from the spline touching the newly created vertex.

    3. Select both "FusedSplineSets_3" and "Vertex_1" together → Type in the Python console:

    MakeFillingFromUnsortedEdges( )

    or:

    mffue( )

Compare Geometries   

  1. Show only the input geometry and the rebuilt face ("FillingFromUnstortedEdges_1"), in shading mode and with different colors.

    2. Select "a380-nose.brep_1" and "FillingFromUnstortedEdges_1" → Right-click > Show Only

    Select "a380-nose.brep_1" and "FillingFromUnstortedEdges_1" → Right-click in the 3D window > Display Mode > Shading

    Select "a380-nose.brep_1" and "FillingFromUnstortedEdges_1" → Type in the Python console:

    SetRandomColors( )

    or:

    src( )

    You can close this study.
