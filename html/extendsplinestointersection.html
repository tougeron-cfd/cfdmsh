ExtendSplinesToIntersection

Short Description   

Extends two splines to intersection points.

Additional Information   

This function first extend input splines straight away to detect possible intersections. I intersection exist, so each spline is extended to this intersection point.

The user can also ask for a straight extension. In this case there are intermediate vertexes between the edge and the target vertex to ensure the extension in straight.

Example   

from cfdmsh import * # To adapt to the cfdmsh installation method

vertex1 = geompy.MakeVertex(-230, -85, 0)
vertex2 = geompy.MakeVertex(-180, 35, 0)
vertex3 = geompy.MakeVertex(-50, 115, 0)
vertex4 = geompy.MakeVertex(-35, 20, 0)
vertex5 = geompy.MakeVertex(-75, -75, 0)
vertex6 = geompy.MakeVertex(-70, -145, 0)

curve1 = geompy.MakeInterpol([vertex1, vertex2, vertex3])
curve2 = geompy.MakeInterpol([vertex4, vertex5, vertex6])
AddToStudy([curve1, curve2], "curve")

extended_spline_wire = ExtendSplinesToIntersection([curve1, curve2])
extended_splines = ExtendSplinesToIntersection([curve1, curve2], single = False)

Conditions of Use   

To detect intersection, both input splines must be as coplanar as possible.
